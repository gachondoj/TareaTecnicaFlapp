# TareaTecnicaFlapp

## Front End

Como correr:

1. Instalar dependencias
````
npm install
````

2. Correr Programa
````
npm run dev
````

## Back End

Como correr:

1. Instalar dependencias
````
npm install
````

2. Agregar llaves de apis Uder y TraeloYa a .env
````
UDER_KEY=[KEY]
TRAELO_KEY=[KEY]
````

3. Correr Programa
````
npm run dev
````
